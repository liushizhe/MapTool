🗂️ Shapefile支持指南

注意： 当前版本的MapTool已经支持Shapefile文件集上传，但需要额外配置shpjs库来完整解析shapefile格式。

📋 Shapefile文件要求

Shapefile是一种矢量数据格式，由多个文件组成。要成功加载shapefile，您需要提供以下文件：

必需文件：

  • .shp - 主文件，包含几何数据
  • .shx - 索引文件
  • .dbf - 属性数据文件

可选文件：

  • .prj - 投影信息文件
  • .cpg - 编码信息文件

🔧 添加shpjs库支持

要完整支持shapefile解析，请按以下步骤操作：

方法1：通过CDN添加（推荐）

在 public/index.html 文件的 <head> 标签中添加：

<script src="https://unpkg.com/shpjs@latest/dist/shp.js"></script>

方法2：通过npm安装

在项目根目录运行：

npm install shpjs jszip --save

然后在 src/main.js 中添加：

import shp from 'shpjs' window.shp = shp

📁 使用方法

  1. 准备完整的shapefile文件集（至少包含.shp、.shx、.dbf文件）
  2. 在地图页面点击"选择Shapefile文件集"按钮
  3. 同时选择所有相关文件（可以按住Ctrl键多选）
  4. 系统会自动检查文件完整性并解析
  5. 解析成功后，数据会显示在地图上
提示： 如果您没有shapefile文件，建议使用GeoJSON格式，它更简单且完全支持。

🔄 替代方案

如果无法配置shpjs库，您可以：

  • 使用在线工具将shapefile转换为GeoJSON格式
  • 使用QGIS等GIS软件导出为GeoJSON
  • 使用ogr2ogr命令行工具转换格式

推荐的在线转换工具：

  • Mapshaper - 功能强大的在线地图数据处理工具
  • MyGeodata Cloud - 支持多种格式转换

  3. 检查浏览器控制台是否有错误信息
